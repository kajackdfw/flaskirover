# Raspi Rover OS

***A Python based Rover OS for the Raspberry PI***

**v0.03.01 WORKING**

1. Flask UI
2. Pi Camera
3. AdaFruit Motor Shield
4. UI Optimized for LG G4

**TESING OR IN DEVELOPMENT**

1. RaspiRover Board v3 support
1. Loadable hardware configs

1. Pillow ( with pip )
1. Flask ( with pip )
